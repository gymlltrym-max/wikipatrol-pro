# WikiPatrol Live Pro


# מה התוסף עושה
ויקיפטרול חי פרו מאפשר ניטור עריכות בזמן אמת בוויקיפדיה ישירות מסרגל הצד. כולל זיהוי השחתות חכם (ORES), מצב כהה, סינון אנונימיים ורשימת מעקב אישית. הכלי האולטימטיבי למנטרים שרוצים יעילות מקסימלית ללא צורך ברענון הדף. אפשר לקרוא או לכתוב ערך ויקיפדיה ולהמשיך לנטר. 
תכונות מרכזיות נוספות:
* ⚡ עדכונים חיים: זרם עריכות שמתעדכן אוטומטית (Live Feed).
* 🎯 סינון מתקדם: אפשרות להציג רק אנונימיים, רק את הגרסה האחרונה, או רק עריכות במרחב הערכים (ללא דפי שיחה).
* 🎨 ממשק חכם: צביעת עריכות חשודות באדום (זיהוי השחתה) ועיצוב Dark Mode מודרני.
* 👁️ תצוגה מקדימה: צפייה בהבדלי גרסאות (Diff) בתוך התוסף עצמו.
* 🔔 מעקב אישי: קבלת התראות על משתמשים ספציפיים שבחרתם.
* 🔒 פרטיות: התוסף אינו אוסף מידע אישי ומשמש ככלי עזר לניטור בלבד

## הוראות התקנה ל-Google Chrome (חינמי) גרסה 1.1.0
מכיוון שהתוסף אינו מפורסם בחנות של גוגל (כדי לשמור עליו בחינם), יש להתקין אותו ידנית:

1. לחצו על הכפתור הירוק **Code** למעלה ובחרו ב-**Download ZIP**.
2. חלצו את קובץ ה-ZIP לתיקייה קבועה במחשב שלכם.
3. פתחו את דפדפן כרום והיכנסו לכתובת: `chrome://extensions`.
4. הפעילו את **"מצב מפתח" (Developer mode)** בפינה הימנית/שמאלית העליונה.
5. לחצו על כפתור **"טעינת פריט Unpacked**. 
6. בחרו את התיקייה שבה שמרתם את קבצי התוסף.
## הוראות התקנה ל-Microsoft Edge (חינמי) גרסה 1.0
1. פתחו את דפדפן Edge והיכנסו לכתובת: `https://microsoftedge.microsoft.com/addons/detail/wikipatrol-live-pro/fkfcbdpcccepfkglfhibigbhinogiegh` 
2. תלחצו על קבל
